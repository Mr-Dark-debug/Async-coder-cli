Search the web for up-to-date information. Returns formatted search results with titles, URLs, and snippets.

Supports multiple backends (configured via websearch.provider in async-coder.json):
- duckduckgo (default, no API key required)
- tavily (TAVILY_API_KEY - 1k free searches/month)
- brave (BRAVE_API_KEY - 2k free queries/month)
- google (GOOGLE_API_KEY + GOOGLE_CSE_ID - 100 free queries/day)
- exa (EXA_API_KEY, experimental)

Use this tool when you need current information, documentation, or context beyond your training data.

The current year is {{year}}. You MUST use this year when searching for recent information or current events.
